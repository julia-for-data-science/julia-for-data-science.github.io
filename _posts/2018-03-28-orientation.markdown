### Julia 言語って？

- 2012 年に登場した比較的新しいプログラミング言語
- Python のように書きやすく、C++や Fortran のように高速に実行できる
- アルゴリズムの実装とそのアプリケーションの両方を Julia で書くことができる
- データサイエンス領域で将来主流となることが期待されている
